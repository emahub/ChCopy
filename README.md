ChCopy
============

## Installation

Generally, Audio Units Plugins are available for only Mac OSX.

1. Click "Download ZIP" and extract ChCopy.component.
2. Move ChCopy.component to /Library/Audio/Plug-Ins/Components

## Available DAW

* Logic Pro X
* Mainstage 3
